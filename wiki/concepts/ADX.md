# ADX — Average Directional Index

## O que mede
Força da tendência (não direção). 0–100.

## Interpretação
| Valor | Interpretação |
|-------|---------------|
| < 20 | Sem tendência (evitar entradas tendência) |
| 20–25 | Tendência fraca emergindo |
| > 25 | Tendência confirmada |
| > 40 | Tendência forte |
| > 60 | Tendência extrema (raro) |

## Uso no Sistema
- ADX > 25 = filtro de entrada habilitado
- ADX > 25 + estrutura bearish 4H = hard block em LONG
- ADX < 20 = aguardar definição antes de entrar
