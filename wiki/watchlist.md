# Watchlist Pessoal

Esta é a lista principal de observação (watchlist). 
Quando o usuário pede para "verificar a watchlist", o agente deve ler este arquivo e executar um scan de cada ativo listado.

| BTC   | BTC    | Market leader               |
| ETH   | ALTCOIN| Smart Contracts leader      |

> **Instruções para o Usuário:** 
> - Adicione ou remova linhas na tabela acima.
